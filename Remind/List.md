## 기본
``` python
a = [1,2,3,[1,2,3]]

a[0] = 1
a[3] = [1,2,3]
or
a[-1] = [1,2,3]
a[3][3] = 3

```

### 비어있는 list는 ``` a = list()```로 생성

## 리스트 연산하기

### 리스트 더하기
``` python
    a = [1, 2, 3]
    b = [4, 5, 6]
    a + b
    [1, 2, 3, 4, 5, 6]

```

### 리스트 곱하기
``` python
    a = [1, 2, 3]
    a *3
    [1, 2, 3, 1, 2, 3, 1, 2, 3]

```

### 리스트 삭제하기
``` python
    a = [1, 2, 3, 4, 5]
    del a[2:]
    a
    >> [1, 2]

```
